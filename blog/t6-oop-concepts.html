What are Variables?

Straight up. There is no other way to digest this. Variables are the memory locations holding data that can be used in the program. A variable can be seen as a place to store data. All variables are unique and they refer to a specific object located in memory and they do this by pointing to that object in memory wich is the famous definitions of "pointers."

variable=5 -> To access the variable it has a pointer that stores [5] in memory.

Type of variables?

There 5 types of variables in ruby that are:

  • CONSTANT=5
  • $global=5
  • @@class=5
  • @instance=5
  • local=5

Constant variables are like normal varibles but it's value is supposed to remain the same thru out the whole program. Global variables can be accesed from anywhere in the program. Class varaibles exist over ALL instances of a class it is like saying that all instances from that class have something in common. Instance varaibles are variables that belong to each instance of the class. Local variables can be accessed on the block where it was defined only.

How to use variables

Let's take for example local variables.

def information
  name=andres
  age=andres
  info=[name,age]
  return info
end

They can be accessed only in the information method scope. We can se here how we return the array that we assigned a variable with the name of info.
