What is an Enumerable?

To be concise an Enumerable is a ruby module that provides various methods to traverse, iterate, go thru each element an manipulate the elements in a structure like an array or a hash.

Let's go deeper

An "enumerator" is the tool we can use to get each element out of a collection in this way. Before we go ahead, it's important to understand what Enumerable and Enumerator are in Ruby terms.

ENUMERABLE VS ENUMERATOR (...coming up)

So what are enumerables for then? They are used to modify the elements in a data structure or have some functionality done with them, some of them or one of them. It like aplying functionality to a container of objects that you own. Let's say you have a container of ten (10) cars with different colors. You want to use the blue car today so you use an enumerable to go thru them and get the blue car for you!

Enumerable#Cycle

So for the enumerable example we are using the "cycle" method. This enumerable gives you the ability to iterate thru the structure as many times as you want. So for example if we say cycle(3) we would be going thru the structure a total of three (3) times.

array = ["Hey", "there", "Coder"]

array.cycle { |x| puts x } # print, hey, there, Coder, hey, there, Coder,hey,there.. forever.

array.cycle(2) { |x| puts x } # print,hey, there, Coder
